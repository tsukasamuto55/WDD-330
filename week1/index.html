Master mobile UX

Lesson 1, Step 1: Mobile First UX

Things to remember

  • Desktop computer sales have been declining since 2011
  • You don't see people with their heads in laptops
  • Start small and scale up.
  • Mobile is not a fad. It will last for a long time.

Lesson 1, Step 2: How to improve UX by understanding Users

SPEED MATTERS

  • Google: 1/2 second delay for search = 20% drop in traffic
  • Bing: 2 second delay for search = 4% drop in revenue
  • Aol.: Fastest 10% of users stay 50% longer.

How to improve Speed

  • Reduce image usage throughout sites/apps
  • AVG/Webfonts are lower bandwidth
  • Optimize and minify CSS and JS files
  • GZIP files to eliminate unnecessary data

Lesson 1, Step 4: Designing Intuitive and Mobile Friendly Forms

  • Ditch the labels, instead using float labels inside of each input.
  • Use a larger checkbox
  • Give a user option to fill out.
  • Pagination and progress indicator are good ideas for a large form.
  • Enabling an auto-fill function.
  • Show password as default and gives a user to choose to hide it or not.
  • When taking numbers, ensure the number keypad is used on mobile.
  • Linking Phone Numbers on your website makes it much easier for people to call from mobile devices.
    Ex. <a href="tel:14035551234">403-555-1234</a>
